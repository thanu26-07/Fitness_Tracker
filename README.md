# 🏋️ AI-Powered Fitness Tracker

An AI-powered full-stack fitness tracking web application that helps users monitor their health by tracking daily calorie intake, calories burned, BMI, and fitness goals. The application also integrates Google Gemini AI to analyze food images and estimate calorie values automatically.

---

## 🚀 Features

- 🔐 Secure User Authentication (Sign Up & Login)
- 👤 User Onboarding & Profile Management
- 🍽️ Food Logging with Calorie Tracking
- 🤖 AI-Based Food Image Analysis using Google Gemini AI
- 🏃 Activity Tracking & Calories Burned
- 📊 Dashboard with Progress Visualization
- 📱 Responsive Design for Mobile & Desktop
- 🌙 Light & Dark Mode Support

---

## ⚙️ Installation

### Clone the repository

```bash

```

```bash

npm run dev
```

### Backend

```bash
cd server
npm install
npm run develop
```
